HTML

Hyper Text Markup Language (HTML) is the dominant markup language developed for displaying web pages. Additionally, HTML provides a way to create structured documents with links, citations, and other items, as well as representing structural meaning for text such as headings, paragraphs, lists, etc.

And it can be used to create interactive forms by embedding images and objects. HTML is written in the form of HTML elements that are "tags" enclosed in angle brackets within the web page content. HTML can contain or load scripts, such as JavaScript, which affects the behavior of HTML processors such as web browsers, and CSS, which defines the appearance and layout of the body and other items. The W3C, co-responsible for the HTML and CSS standards, recommends the use of CSS for clear, expressive markup
